# 💞 ¿Quieres ser mi Novia?

Una página web interactiva y divertida para pedirle a alguien especial que sea tu novia. Con animaciones, modales, burbujas y un botón "NO" que... digamos que es escurridizo 😉

## 🚀 Demo

Abre `index.html` en tu navegador o despliega en GitHub Pages / cualquier hosting estático.

## ✨ Características

- Diseño responsivo (adaptado a móviles y desktop)
- Botón "SI" que abre un modal con palabras bonitas
- Botón "NO" que huye del cursor
- Modal con imagen meme
- Fondos animados con gradiente y burbujas
- Audio de ambiente
- Animaciones neon y parpadeo

## 🛠️ Personalización

- Edita `index.html` para cambiar los mensajes en los modales
- Reemplaza las imágenes en `img/` por las tuyas
- Cambia el audio en `media/audio.mp3`
- Ajusta colores y estilos en `src/styles.css` y `src/modal.css`
